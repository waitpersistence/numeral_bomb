Numeral Bomb RL (数字炸弹强化学习项目)

本项目实现了一个基于 深度 Q 网络 (DQN) 的强化学习智能体，用于破解经典的“数字炸弹”（猜数字 / Bulls and Cows）游戏。该游戏要求智能体通过最少的猜测次数锁定一个四位各不相同的目标数字。

🚀 项目亮点高性能状态管理：利用 $5040 \times 5040$ 的预计算反馈矩阵，将逻辑排除转化为高效的矩阵掩码运算。信息熵减策略：智能体通过训练自动学习如何最大化每一轮的“信息增益”。双向博弈模式：支持人类与 AI 的实时对战，AI 兼具进攻（猜测）与防御（裁判）功能。

📂 文件结构
env.py: 核心环境类，包含游戏规则实现、预计算反馈矩阵及状态更新逻辑。
train.py: DQN 算法实现。包含 Q 网络架构、经验回放池及模型训练循环。
play.py: 测试脚本，AI 作为进攻方猜测玩家脑海中的数字。battle.py: 完整的双向对战模式，人类与 AI 轮流猜测。🎮 使用方法1. 训练 AI运行训练脚本。初次运行会生成反馈矩阵（约需 2-5 秒）。训练结束后会保存 number_bomb_agent.pth。Bashpython train.py
2. 开始对战确保目录下已有训练好的 .pth 模型文件，运行对战脚本：Bashpython battle.py
🧠 技术细节
1. 状态表示 (State Representation)状态被建模为一个 $5040$ 维的二进制向量 $S \in \{0, 1\}^{5040}$。1：表示该索引对应的数字仍有可能是正确答案。0：表示该数字已被逻辑排除。2. 动作掩码 (Action Masking)在推理阶段，为了防止 AI 猜测已被排除的数字，我们引入了动作掩码技术：$$Q_{masked} = Q(s, a) + (S - 1) \times 10^9$$这确保了智能体永远只在当前的候选集空间内选择最优动作。3. 反馈矩阵预计算的矩阵 $M_{i,j}$ 存储了所有猜测对目标数字的匹配结果，极大地提升了训练时的样本吞吐量，将 $O(N)$ 的过滤操作降低到了向量化运算级别。
